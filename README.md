## 🚀 Featured Project

### 🏨 [Restful Booker — API Test Automation Framework](https://github.com/AutoByDolby/RestfulBooker)

> An end-to-end REST API automation framework built with **Java**, **Rest Assured**, **Cucumber BDD**, and **JUnit** — covering the full booking lifecycle of a hotel booking API.

- ✅ Full CRUD coverage — CreateBooking, GetBooking, UpdateBooking, PartialUpdateBooking, DeleteBooking
- ✅ Auth token generation and injection for protected endpoints
- ✅ BDD Gherkin scenarios with Cucumber and JUnit
- ✅ Hooks-based pre-scenario setup to avoid redundant API calls
- ✅ Reusable assertion helpers and centralized config via `global.properties`
